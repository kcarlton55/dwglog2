Add a new record:

Press the button that has the plus sign on it. Enter the part no. and description. Other record fields (dwg no., date, author) will be automatically filled in. If only the first four digits of a part no. are entered, then once the part no. is submitted the remainder of the part no. will be appended automatically. For example 0300 will turn into 0300-2020-401.

After you enter the first four digits for a part number, if the number is for a common part type, the description field will fill in automatically with a partial description. Finish filling in the description field or alter it accordingly. For example, entering 0300 in the pt. no. field will cause the word BASEPLATE to show in the description field.

Search examples:

To search, enter a query into the box provided and push the Enter key.

  query:

  • 2730-2020-8*

  yields:

  • Dwg. no.Part no.DescriptionDateAuthor
    20208702730-2020-870BRACKET PANEL 20X20 200GAL TANK CS12/04/2020tasbury
    20208632730-2020-863PANEL BRACKET 16x14 CS12/02/2020whoyt
    etc.

  query:

  • 1*/*/2020; kcarlton

  yields:

  • Dwg. no.Part no.DescriptionDateAuthor
    20208590300-2020-859BASEPLATE FOR DV0016B-DA2 W/MONO BLK MTR12/01/2020kcarlton
    2020857094411DV0016B-DA2 W/MONO BLK MTR & BASE11/27/2020kcarlton
    etc.

Here found are records (rows of data) created in November or December (i.e., 1*/*/2020, where 1* = Nov or Dec, and the subsequent * = any day) and where only kcarlton is the author. Thus other authors are eliminated allowing a clear look about a particular author.

How was this search accomplished? The ; acts as a special character. It finds the intersection of search groups so that only elements that each search group has in common are output. In this case the two search groups are 1*/*/20 and kcarlton.

  query:

  • 09*; kcar*; 11/*/2020   or   09*; rcol*; 11/*/2020   or   09*; wh*; 11/*/2020

  yields:

  • Dwg. no.Part no.DescriptionDateAuthor
    2020857094411DV0016B-DA2 W/MONO BLK MTR & BASE11/27/2020kcarlton
    2020855094379DVW0303KP1-35 230/460V TEFC11/25/2020rcollins
    2020823094020VMX0753KA1-00 460V TEFC11/13/2020whoyt
    etc.

The "or" keyword means that if any or all of the "2nd stage" groups (groups with/without a ; characater) have results, then show any or all of those results. The word "or" must be in lower case letters.

Note that searches are case sensitive. That is *rc* and *RC* will yield different results. For more information about searching, see: https://en.wikipedia.org/wiki/Glob_(programming)

Update a field:

To update a field (a table cell), change the text in the field and then press the Enter key.

If in a part no. field you put the first for digits and a dash (e.g. 0300-), the part no. will autogenerate (e.g. 0300-2021-030). For changing a drawing no., see Overwrite a drawing number below.

Delete a record:

To delete a data record (a table row), enter one of the following words into the field that contains the drawing number: delete, remove, del, or rm. Then press Enter.

Refresh the table:

While you are working on the dwglog2 program, other users simultaneously have access to the same data shown to you. You may wish to push the refresh button to see changes other users have made while you have been working with the program. Also, if you wish to assure that your data has been updated successfully, push the refresh button. The most recent data will be pulled from the database.

AutoCopy:

Automatically copy contents of a clicked field (cell) to the clipboard. That is, Ctrl-C not needed.

Reserve a block of drawing numbers:

Perhaps you would like to have a consecutive group of drawing numbers for your project. To do that, first add a new part number. Then to that new drawing number that appears, increase its value the desired amount. A gap of drawing numbers will now exist below your new number. It is in that gap that you will place your new numbers. To do that, add another new number and then change that drawing number to a number that will fit in the gap. When you do that, you'll see the list of dwg. nos. update and you'll see that your new number has been relocated to be in the correct order.

Note: DO NOT change a drawing number that you did not create unless you consult the person that created the number. That number may have already been placed on a drawing.

Overwrite a drawing number:

Depending on company policy, you can assign a different drawing number other than what is generated by the program. For example, if an existing drawing number is 2020866, and it is desired to make it better associate with a drawing for Atlas Copco, the number could be changed to AC20866, or to some other scheme. To do this, simply replace the drawing number with the number you want. If you want to go back to the orignally assigned drawing number, edit the drawing number field and leave the field blank. The original, default number will reappear.
