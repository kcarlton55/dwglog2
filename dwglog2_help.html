Add a new record:

Press the button that has the plus sign on it. Enter the part no. and description. Other record fields (dwg no., date, author) will be automatically filled in. If only the first four digits of a part no. are entered, then the remainder of the part no. will be filled in automatically. For example 0300 renders 0300-2020-401.

If the first four numbers and a dash, -, are entered, for example 0300-, then the description field will be filled with a partial description for you to complete. For example 0300- will yield BASEPLATE, and 6405- will yield CONDENSATE TANK. Then pressing Enter will generate a drawing number, e.g. 0300-2020-401.

Search examples:

To search, enter a query into the box provided and push the Enter key.

  query:

  • VMX0036*

  yields:

  • Dwg. no.Part no.DescriptionDateAuthor
    2020808093954VMX0036MA1-00 460V TEFC11/04/2020whoyt
    2020804093859VMX0036KA1-00 575V TEFC11/04/2020kcarlton
    etc.

  query:

  • 09*; 0[345]/*/2020; kcarlton

  yields:

  • Dwg. no.Part no.DescriptionDateAuthor
    2020811093902VMX0103KA1-00 460V TEFC03/25/2020kcarlton
    2020803093840RVL212HH-14 W/OPTIONS 380V/50/305/19/2020kcarlton
    etc.
Where character groups are  09*,  0[345]/*/2020,  and  kcarlton. Each group is searched individually. The ; charater then tells the program to show only results that all three groups have in common. The above example yields production units (i.e. 09*) of March, April, or May (i.e. months 3 or 4 or 5) of 2020, and created by kcarlton. In other words, the ; character finds the intersection of search results of the three groups.

  query:

  • 09*; kcar*; 11/*/2020   or   09*; rcol*; 11/*/2020   or   09*; wh*; 11/*/2020

  yields:

  • Dwg. no.Part no.DescriptionDateAuthor
    2020818093798VMXVFD0203KA2-00 460V TEFC11/11/2020kcarlton
    2020816094189RVL031H-03 PUMP W/VG 208-230/460V11/11/2020rcollins
    2020808093954VMX0036MA1-00 460V TEFC11/04/2020whoyt
    etc.

The "or" keyword means that if any or all of the intersection groups (groups with/without a ; characater) have results, then show any or all of those results. The word "or" must be in lower case letters.

Note that searches are case sensitive. That is *rc* and *RC* will yield different results. For more information about searching, see: https://en.wikipedia.org/wiki/Glob_(programming)

Update a field:

To update a field (a table cell), change the text in the field and then press the Enter key.

Delete a record:

To delete a data record (a table row), enter one of the following words into the field that contains the drawing number: delete, remove, del, or rm. Then press Enter.

Refresh the table:

While you are working on the dwglog2 program, other users simultaneously have access to the same data shown to you. You may wish to push the refresh button to see changes other users have made while you have been working with the program. Also, if you wish to assure that your data has been updated successfully, push the refresh button. The most recent data will be pulled from the database.

AutoCopy:

Automatically copy contents of a clicked field (cell) to the clipboard. That is, Ctrl-C not needed.

Reserve a block of drawing numbers:

Perhaps you would like to have a consecutive group of drawing numbers for your project. To do that, first add a new part number. Then to that new drawing number that appears, increase its value the desired amount. A gap of drawing numbers will now exist below your new number. It is in that gap that you will place your new numbers. To do that, add another new number and then change that drawing number to a number that will fit in the gap. When you do that, you'll see the list updates and your new number will relocate so that it is the correct order.

Overwrite a drawing number:

Depending on company policy, you can assign a different drawing number other than what is generated by the program. For example, if an existing drawing number is 2020866, and it is desired to make it better associate with a drawing for Atlas Copco, the number could be changed to AC20866 or some other scheme. To do this, simply replace the drawing number with the number you want. If you want to go back to the orignally assigned drawing number, simply edit the number and leave the field blank. The original, default number will reappear.
